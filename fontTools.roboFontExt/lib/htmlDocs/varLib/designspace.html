designspace¶

Rudimentary support for loading MutatorMath .designspace files.

fontTools.varLib.designspace.load(filename)[source]¶

Load designspace from a file name or object. Returns a dictionary containing three (optional) items: - list of “axes” - list of “sources” (aka masters) - list of “instances”

fontTools.varLib.designspace.loads(string)[source]¶

Load designspace from a string.
